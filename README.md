# Ossino FE

To run the front-end with a fully working live chat, follow these steps:

1. Navigate to the WebSocket server directory:
   `cd demo-server/`
2. Install the dependencies for the WebSocket server:
   `npm i`
3. Return to the root directory:
   `cd ..`
4. Install the front-end dependencies:
   `npm i`

### IMPORTANT!

To successfully run server & front-end in sync, the `concurrency` package should be installed.
Please check that ``concurrency` is installed after cloning a project from repo.
